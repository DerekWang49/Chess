Ensure that Rectangle.java, Triangle.java, and Circle.Java are all in the same directory as FractalDrawer.java
Navigate to the directory containing FractalDrawer.java and type the command: "javac FractalDrawer.java".
Then type in "java FractalDrawer". It will then prompt you to choose one of the following shapes: circle, triangle,
or rectangle. Simply type in the desired shape and click on the java icon that appears on the dashboard. The shapes
will then appear within the java window. The program will only run if the user's device is able to output any visuals
from java or run java programs. As of now, there are no known bugs or defects in the program
- For finding the list of available colors within java.awt.Color: https://docs.oracle.com/javase/7/docs/api/java/awt/Color.html
- For implementing java.util.Random: https://stackoverflow.com/questions/4246351/creating-random-colour-in-java
